💡 Setup Instructions

Each agent needs two environment variables set in Railway:

VAULT_URL — The internal URL of this vault service (e.g., http://vault.railway.internal:8080)
VAULT_TOKEN — The agent's unique token from the table above

Agents use these to fetch API keys securely without ever seeing the raw values in their environment variables.
